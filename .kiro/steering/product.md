# Product

This is a micro-blogging web application — similar in concept to Twitter/X. Users can register, log in, create short posts (up to 280 characters), like posts, follow/unfollow other users, and view a feed of posts.

## Core Features
- User registration and authentication (via AWS Cognito)
- Create, view, and like posts (280 character limit)
- Follow/unfollow users
- User profiles with follower/following counts
- Paginated post feed with sorting (newest/popular)
